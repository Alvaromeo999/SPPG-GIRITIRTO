Lampiran III: Matrix Standard Operating Procedure (SOP)

Sistem Informasi & Website Terpadu SPPG Giritirto

ALUR KERJA & FUNGSI SISTEM "GIRITIRTO GIZI HUB"
AKTOR (SIAPA) MODUL / FITUR ALUR PROSES (BAGAIMANA) OUTPUT / TUJUAN
ADMIN SPPG
(Operator) 		1. Manajemen Menu
(Input Data) 		Login Admin ‚Üí Input Nama Menu ‚Üí Input Nilai Gizi ‚Üí Upload Foto ‚Üí Publish. Wali murid mengetahui menu & gizi H-1 atau Mingguan.
2. Laporan Distribusi
(Update Harian) 		Cek Data Dapur ‚Üí Update Angka Porsi (Dashboard) ‚Üí Upload Foto Kegiatan Hari Ini. Transparansi bahwa makanan benar-benar tersalurkan.
3. Procurement
(Belanja Bahan) 		Cek Stok Gudang ‚Üí Posting "Info Kebutuhan Bahan" di Web ‚Üí Menunggu Respon Mitra Tani. Membuka peluang bagi petani lokal untuk menyuplai.
MITRA TANI
(Warga Lokal) 		1. Suplai Bahan Baku
(Respon Ekonomi) 		Buka Website (Menu Mitra) ‚Üí Cek Daftar Kebutuhan ‚Üí Klik Tombol WA ‚Üí Ajukan Stok & Harga. Ekonomi berputar di desa (Petani mendapat pasar pasti).
WALI MURID
(Masyarakat) 		1. Monitoring Gizi
(Pengawasan) 		Buka Website ‚Üí Lihat Menu Hari Ini ‚Üí Cek Detail Protein/Kalori. Ketenangan orang tua terhadap asupan gizi anak.
2. Aspirasi / Aduan
(Feedback) 		Temukan Masalah ‚Üí Klik Menu "Lapor/Aduan" ‚Üí Kirim Pesan via Sistem/WA. Perbaikan kualitas layanan SPPG yang berkelanjutan.
KEPALA SPPG
/ LURAH 		1. Pengawasan
(Controlling) 		Buka Website (Kapanpun/Dimanapun) ‚Üí Cek Grafik Distribusi & Foto Laporan Harian. Memastikan program berjalan sesuai Juknis BGN Pusat.
* SOP ini dirancang untuk meminimalisir input manual yang berulang dan memaksimalkan transparansi publik secara digital (Paperless).
